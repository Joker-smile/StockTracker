# StockTracker (潜行盯盘工具)

StockTracker 是一款基于 C# WinForms 开发的**极简、高度隐蔽**的桌面盯盘软件，特别为办公室摸鱼和需要低调关注股市行情的人士设计。它不仅提供了基础的实时行情展示，还内置了**专门针对A股特征优化的“盘口智测”量化模型**，通过实时结合量价、均线与 K 线形态对后市走势进行推演。

## ✨ 核心特性

### 1. 极致隐蔽的沉浸式 UI
- **无感存在**：无标题栏、无边框、不在系统任务栏显示图标。
- **水印级显示**：采用半透明黑色背景与暗色字体，完美融入深色编辑器或深色壁纸背景中，仿佛屏幕上的系统水印。
- **自由悬浮**：支持鼠标左键按住拖动到任意屏幕角落。

### 2. 极简交互与配置
- **右键菜单管理**：在界面上点击鼠标右键即可呼出控制菜单（添加股票、清空列表、退出程序）。
- **智能防呆输入**：添加自选股时**只需输入 6 位纯数字代码**（如 `000001` 或 `600519`），核心引擎会自动推断出对应的股票市场前缀（sh/sz/bj）和所属板块。
- **本地持久化**：自选股列表自动保存在本地 `stocks.txt`，下次打开自动加载。

### 3. A股定制版“智测”量化引擎 (2.0 版)
不仅是量量，更是基于 A 股盘口特征的实时逻辑分析，已针对大 A 交易环境深度优化：
- **⏳ 动态量比归一化 (Annualized Ratio)**：根据当前交易时间自动估算全天成交量，不再在早盘误判“缩量”。
- **📊 动态多板块阈值**：自动识别主板 (10%)、科创/创业板 (20%)、北交所 (30%) 涨跌停限制，精准判断打板/跌停状态。
- **📈 趋势感应 (Trend-Aware)**：结合近 5 日趋势斜率，精准区分“超跌反弹”与“趋势主升”。
- **📉 多维 K 线形态**：识别“仙人指路”、“金针探底”、“避雷针”等 A 股典型盘口形态，引入十字星过滤逻辑防止误报。
- **🛡️ 5 日线回踩逻辑**：以 MA5 偏离度带（Near-MA5 Band）为基准，判断“稳步攀升”还是“破位杀跌”。
- **🚀 高性能缓存**：历史 K 线数据 30 分钟缓存，确保每 3 秒刷新时极速响应且不触发 API 限流。

## 🚀 部署与运行

本项目使用了 .NET 9（其他 .NET Core 版本也能向下兼容编译）。

### 编译单文件绿色版
您可以直接在项目根目录下通过 `dotnet cli` 将其打包成一个**无需目标系统安装任何 .NET 环境即可运行**的单执行文件（Exe）：

```bash
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true -o publish
```

编译完成后，去 `publish` 文件夹提取 `StockTracker.exe` 即可拷贝到任何 Windows 64位电脑上直接双击运行！
