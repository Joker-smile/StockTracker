## 发现的问题及修复

### 问题1: 日期字段解析错误 ❌ → ✅

**错误信息**:
```
System.FormatException: The input string '2025-02-26' was not in a correct format.
```

**根本原因**:
K线数据第一个字段是日期字符串（如"2025-02-26"），代码尝试将其解析为`double`导致异常。

**修复方案**:
```csharp
// 修改前（错误）
opens.Add(double.Parse(parts[0]));  // ❌ parts[0]是日期

// 修改后（正确）
// 跳过parts[0]（日期），从parts[1]开始解析
if (double.TryParse(parts[1], out double openVal))
    opens.Add(openVal);
if (double.TryParse(parts[2], out double closeVal))
    closes.Add(closeVal);
// ... 其他字段
```

**修复位置**:
- `MainWindow.axaml.cs:705-735` - 个股K线解析
- `MainWindow.axaml.cs:546-556` - 大盘K线解析

---

### 问题2: 数据解析不一致风险 ❌ → ✅

**问题描述**:
使用多个独立的`if`语句解析字段，如果某个字段解析失败，会导致所有列表长度不一致，后续计算出错。

**修复方案**:
```csharp
// 修改前（有风险）
if (double.TryParse(parts[1], out double openVal))
    opens.Add(openVal);  // 可能添加
if (double.TryParse(parts[2], out double closeVal))
    closes.Add(closeVal);  // 可能添加，导致列表长度不一致

// 修改后（安全）
// 确保所有必需字段都能成功解析，否则跳过这条K线
if (!double.TryParse(parts[1], out double openVal) ||
    !double.TryParse(parts[2], out double closeVal) ||
    !double.TryParse(parts[3], out double highVal) ||
    !double.TryParse(parts[4], out double lowVal) ||
    !double.TryParse(parts[5], out double volVal) ||
    !double.TryParse(parts[8], out double pctVal))
{
    continue; // 跳过解析失败的K线
}

// 所有字段都成功解析，统一添加
opens.Add(openVal);
closes.Add(closeVal);
highs.Add(highVal);
lows.Add(lowVal);
vols.Add(volVal);
pcts.Add(pctVal);
```

**修复位置**:
- `MainWindow.axaml.cs:705-735`

### 问题3: 网络请求脆弱性 ❌ → ✅

**错误信息**:
```
System.Net.Http.HttpIOException: The response ended prematurely.
```

**根本原因**:
- 东方财富API检测到频繁请求，主动断开连接（防爬虫机制）
- 网络不稳定导致连接中断
- 服务器负载高，响应超时

**修复方案**:
新增带重试机制的HTTP请求方法：

```csharp
private async Task<string> HttpGetWithRetryAsync(string url, int maxRetries = 3)
{
    for (int i = 0; i < maxRetries; i++)
    {
        try
        {
            var response = await _httpClient!.GetStringAsync(url);
            return response;
        }
        catch (HttpRequestException ex) when (i < maxRetries - 1)
        {
            // 网络错误，等待后重试
            await Task.Delay(1000 * (i + 1)); // 1s, 2s, 3s递增延迟
            Program.LogError($"HTTP request failed, retry {i + 1}/{maxRetries}", ex);
        }
    }
    return "";
}
```

**特点**:
- ✅ 智能重试：最多重试3次
- ✅ 递增延迟：1秒 → 2秒 → 3秒
- ✅ 异常捕获：只捕获HttpRequestException
- ✅ 日志记录：记录每次重试信息

**应用位置**:
1. `CheckMarketEnvironmentAsync` - 大盘环境检测
2. `CheckTechnicalAndMomentumAsync` - 个股K线获取
3. `GetBaseStocksAsync` - 候选池列表获取
