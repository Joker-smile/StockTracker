# StockTracker 智测计算规则与提示语说明 (v3.0)

## 一、智测核心逻辑概述

智测系统通过分析股票的实时盘口数据，结合历史均线和量能指标，对当前股票状态进行**定性分析**，给出"看多"、"观察"或"看空"的判断，并附上具体的盘口形态描述。

**v3.0更新**: 智测系统与选股系统解耦，专注于实时盘口分析，为已持有的股票提供动态监控信号。

---

## 二、核心计算维度

### 1. 时间加权量比 (Volume Ratio)

**计算公式：**
```
原始量比 = 当前分钟成交量 / 过去5日平均成交量
时间进度 = 已交易分钟数 / 240 (全天交易分钟数)
量比 = 原始量比 / 时间进度
```

**特殊处理：**
- 开盘前15分钟量比通常虚高，采用平滑处理：`平滑进度 = 时间进度 × 0.5 + 0.05`
- 时间段划分：
  - 集合竞价：9:15-9:25
  - 早盘：开市后30分钟内
  - 尾盘：14:30之后
  - 午中：其他时段

### 2. MA5 乖离率 (MA5 Bias)

**计算公式：**
```
MA5 = 最近5日收盘价平均值
MA5乖离率 = (当前价格 - MA5) / MA5 × 100%
```

**判断标准：**
- `乖离率 > 12%` 或 `近期5日涨幅 > 20%` → 处于高位
- `乖离率 < 2%` → 均线附近
- `价格 > MA5` → 多头区间
- `价格 < MA5` → 空头区间

### 3. 日内K线解剖

**关键指标：**
```
实体顶部 = MAX(开盘价, 当前价格)
实体底部 = MIN(开盘价, 当前价格)
实体大小 = 实体顶部 - 实体底部
上影线 = 最高价 - 实体顶部
下影线 = 实体底部 - 最低价
```

**影线判定：**
- 长上影线：`上影线 > 实体大小 × 2` 且 `上影线 > 昨收 × 2.5%`
- 长下影线：`下影线 > 实体大小 × 2` 且 `下影线 > 昨收 × 2.5%`

## 三、智测判定池与提示语

### 【优先级1】极端行情判定

| 触发条件 | 提示语 | 说明 |
|---------|--------|------|
| 触及涨停价 + 尾盘 + 量比>2.2 | **[涨停]分歧封板 (尾盘)** | 尾盘涨停但分歧较大 |
| 触及涨停价 + 尾盘 + 量比≤2.2 | **[涨停]稳稳封死 (尾盘)** | 尾盘强势封死 |
| 触及涨停价 + 量比<0.6 | **[涨停]一字板** | 极度控盘，缩量封板 |
| 触及涨停价 + 量比>4.0 | **[涨停]分歧烂板** | 爆量打板，分歧巨大 |
| 触及涨停价 + 其他 | **[涨停]强势封板** | 正常封板 |
| 最高曾触涨停 + 现价回落 | **[警惕]炸板回落** | 封板失败，资金分歧 |
| 触及跌停价 + 量比>1.2 | **[跌停]放量杀跌** | 跌停但仍有承接 |
| 触及跌停价 + 量比≤1.2 | **[跌停]缩量封死** | 一字跌停，无承接 |

---

### 【优先级2】高位风险预警

| 触发条件 | 提示语 | 说明 |
|---------|--------|------|
| 高位 + 量比>2.5 + 涨幅<2% | **[风险]高位放量滞涨** | 主力可能出货 |
| 高位 + 长上影线 | **[风险]高位见顶回落** | 抛压巨大 |
| 高位 + 下跌 | **[警惕]高位派发中** | 资金流出 |
| 跌破MA5 + 量比>1.8 + 跌幅>2% | **[风险]放量破位** | 支撑失效，加速下跌 |

---

### 【优先级3】经典K线形态

| 触发条件 | 提示语 | 说明 |
|---------|--------|------|
| 长上影 + 非高位 + 上涨 | **[看多]回落蓄势** | 仙人指路，试盘蓄势 |
| 长上影 + 高位或下跌 | **[观察]冲高回落** | 抛压较重 |
| 长下影 + 价格>MA5 | **[看多]深V反弹** | 金针探底，支撑有效 |
| 长下影 + 价格≤MA5 | **[观察]谷底支撑** | 下方有承接 |

---

### 【优先级4】量价共振分析（核心判定）

#### 多头区间（价格 > MA5）

| 触发条件 | 提示语 | 说明 |
|---------|--------|------|
| 量比>2.0 + 涨幅>3.5% + 非高位 | **[看多]放量上攻** | 资金积极做多 |
| 量比>2.0 + 涨幅>3.5% + 高位 | **[警惕]高位放量** | 追高风险较大 |
| 量比<0.35 + 涨幅>0.5% | **[看多]缩量稳涨** | 锁仓良好，控盘上涨 |
| 乖离率<2% + 涨幅≥-0.5% | **[趋势]均线支撑** | 均线附近有支撑 |
| 其他多头情况 | **[趋势]震荡攀升** | 稳步上升 |

#### 空头/整理区间（价格 ≤ MA5）

| 触发条件 | 提示语 | 说明 |
|---------|--------|------|
| 量比<0.3 | **[盘整]缩量筑底** | 地量，等待方向选择 |
| 跌幅<-3% + 量比>1.3 | **[看空]杀跌动能** | 放量下跌，弱势 |
| 其他情况 | **[盘整]弱势震荡** | 等待企稳 |

## 四、时段标识

- **(早盘)**：开市后30分钟内，波动较大
- **(尾盘)**：14:30之后，关注尾盘偷袭或砸盘
- 无标识：盘中正常时段

---

## 五、涨跌停价格计算

```
涨跌幅限制 =
  - ST/退市股：5%
  - 科创板(688)/创业板(300/301)：20%
  - 北交所(8/4)：30%
  - 其他：10%

涨停价 = 昨收 × (1 + 涨跌幅限制)
跌停价 = 昨收 × (1 - 涨跌幅限制)
```

## 六、智测使用建议

### ✅ 看多信号
- **[看多]放量上攻**：资金积极介入，可跟随
- **[看多]缩量稳涨**：锁仓良好，稳健上涨
- **[看多]回落蓄势**：试盘后可能继续上攻
- **[看多]深V反弹**：底部支撑确认

### ⚠️ 观察信号
- **[观察]冲高回落**：上方有压力
- **[观察]谷底支撑**：等待企稳确认
- **[趋势]均线支撑**：关注均线得失

### ❌ 看空/风险信号
- **[风险]高位放量滞涨**：主力可能出货
- **[风险]放量破位**：支撑失效
- **[警惕]炸板回落**：封板失败
- **[看空]杀跌动能**：下跌趋势中

## 七、智测数据缓存机制

- **缓存时间**：30分钟
- **缓存内容**：历史K线数据（MA5、成交量等）
- **目的**：减少API请求，提高响应速度

## 八、与选股系统的关系 (v3.0)

### 智测 vs 选股

| 维度 | 智测系统 | 选股系统 |
|------|---------|---------|
| **用途** | 监控已持有股票 | 筛选新标的 |
| **数据源** | 实时盘口数据 | 历史K线数据 |
| **更新频率** | 3秒一次 | 手动触发 |
| **输出** | 实时信号提示 | 选股结果列表 |
| **核心指标** | 量比、MA5乖离、K线形态 | MACD、ATR、盈亏比、量能形态 |

### 配合使用建议

1. **选股阶段**: 使用选股系统筛选出高质量标的（胜率78%）
2. **买入后**: 使用智测系统实时监控，根据信号调整持仓
3. **智测信号解读**:
   - `[看多]`信号 → 持有/加仓
   - `[观察]`信号 → 谨慎持有，关注后续走势
   - `[风险]`/`[看空]`信号 → 考虑减仓或止损
