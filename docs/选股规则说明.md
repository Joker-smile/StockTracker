# StockTrader 智能选股规则说明

## 一、选股系统概述

本选股系统采用**纯C#原生实现**的量化选股引擎，无需Python依赖，支持并发扫描全A股市场。选股策略结合**基本面筛选**、**技术面分析**和**买点识别**，精选处于上升趋势且具备优质买点的个股。

---

## 二、选股流程架构

```
┌─────────────────────────────────────────────────────────┐
│                    选股执行流程                          │
├─────────────────────────────────────────────────────────┤
│  1. 大盘环境判断 → 确定当前市场状态                      │
│  2. 获取基础股票池 → 过滤ST/退市/停牌                    │
│  3. 基本面筛选 → 市盈率/市值/换手率                      │
│  4. 技术面筛选 → 均线排列/趋势判断                       │
│  5. 买点识别 → 回踩/突破/多头持有                        │
│  6. 风险过滤 → 连续下跌/放量暴跌/放量滞涨                │
│  7. 概念板块获取 → 标注所属题材                          │
└─────────────────────────────────────────────────────────┘
```

---

## 三、大盘环境判断

### 数据来源
- 上证指数日K线（最近250个交易日）

### 判断标准

| 市场状态 | 判定条件 | 策略调整 |
|---------|---------|---------|
| **多头市场** | 指数 > MA20 且 MA20 > MA60 | 正常选股，积极参与 |
| **空头市场** | 指数 < MA20 | 谨慎选股，提高门槛 |
| **震荡市场** | MA20 ≤ 指数 ≤ MA60 | 中性策略 |

---

## 四、基础股票池获取

### 数据源
东方财富API：`http://82.push2.eastmoney.com/api/qt/clist/get`

### 基础过滤规则
- ❌ 排除：ST股票
- ❌ 排除：退市股票
- ❌ 排除：停牌股票（价格为0）

---

## 五、基本面筛选规则

### 规则1：市盈率筛选
```
PE > 0
```
- ❌ 排除负PE（亏损公司）
- ✅ 说明：不设PE上限，避免错过成长股

### 规则2：市值筛选
```
15亿 ≤ 市值 ≤ 500亿
```
- 排除过小市值（风险大）
- 排除过大市值（弹性小）

### 规则3：换手率动态筛选

根据市值分层设定换手率范围：

| 市值区间 | 换手率范围 | 说明 |
|---------|-----------|------|
| 小盘股 (< 50亿) | 2% ~ 20% | 活跃但不过热 |
| 中盘股 (50-200亿) | 1.5% ~ 15% | 适度活跃 |
| 大盘股 (200-500亿) | 0.8% ~ 12% | 相对稳健 |

**额外过滤：**
```
换手率 > 25% → 排除（可能诱多）
```

## 六、技术面筛选规则

### 数据要求
- 需要最近250个交易日（约1年）的日K线数据
- 不足250天的次新股直接排除

### 核心技术指标

#### 1. 长期趋势判断
```
MA200 = 最近200日收盘价均值
当前价格 > MA200 → 确保长期趋势向上
```

#### 2. 均线多头排列
```
MA5 > MA10 > MA20
```
- 确保短期、中期均线呈多头排列
- 排除处于下跌趋势的股票

#### 3. 连续下跌过滤

**检查最近5个交易日：**

| 过滤条件 | 阈值 | 说明 |
|---------|------|------|
| 连续下跌天数 | ≥ 4天 | 排除连续下跌 |
| 累计跌幅 | < -10% | 排除深度调整 |

## 七、买点识别规则

### 买点类型

| 买点类型 | 触发条件 | 风险等级 |
|---------|---------|---------|
| **回踩买点** | \|MA20偏离度\| ≤ 3% + 缩量 | 低风险 ⭐⭐⭐ |
| **回踩缩量** | 回踩买点 + 近3天持续缩量 | 最优 ⭐⭐⭐⭐ |
| **突破买点** | MA20偏离度 3%~12% + 涨幅>2% + 放量 | 中风险 ⭐⭐ |
| **多头持有** | MA20偏离度 0~6% + 微涨 | 稳健 ⭐⭐⭐ |

### 计算公式
```
MA20偏离度 = (当前价格 / MA20 - 1) × 100%
放量判定 = 当日成交量 > MA20成交量 × 1.3
缩量判定 = 当日成交量 < MA20成交量 × 1.1
```

---

## 八、风险过滤规则

### 规则1：放量暴跌过滤
```
最近10天内，任一日：
  跌幅 < -5% 且 成交量 > MA20成交量 × 1.5
  → 排除
```

### 规则2：放量滞涨过滤
```
针对"突破买点"和"多头持有"：
  最近3天平均成交量 > MA20成交量 × 2.0
  且 最近3天平均涨幅 < 1%
  → 排除（可能主力出货）
```

### 规则3：活跃度要求
```
最近5天内：
  最大换手率 > 4%
  平均换手率 > 2.5%
```

## 十二、选股策略建议

### ✅ 最优买点（推荐）
- **回踩缩量**：风险最低，胜率最高
- **回踩买点**：均线支撑，缩量企稳

### ⚠️ 谨慎参与
- **突破买点**：需要确认突破有效性
- **多头持有**：关注均线支撑

### ❌ 避免操作
- 高位放量滞涨
- 放量暴跌后反弹
- 连续下跌趋势

---

## 十三、风险提示

1. **历史数据不代表未来**：技术分析基于历史，无法保证未来表现
2. **大盘环境变化**：大盘转弱时，选股成功率会下降
3. **停牌风险**：选股时未停牌，买入后可能停牌
4. **数据延迟**：API数据可能有延迟，不是实时行情
