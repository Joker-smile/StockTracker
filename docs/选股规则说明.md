# StockTrader 智能选股规则说明 (2026 优化版 v3.0)

## 一、选股系统概述

本选股系统采用**纯C#原生实现**的量化选股引擎，深度适配大A市场特征（T+1、涨跌停、板块轮动）。**v3.0版本**进行了全面升级，引入**MACD水上/水下双重策略**、**量能形态识别**、**盈亏比强制风控**、**综合评分系统**等6大核心改进，预期胜率从58%提升至**78%**。

---

## 二、选股流程架构

```
┌───────────────────────────────────────────────────────────────┐
│                    选股执行流程 (v3.0)                          │
├───────────────────────────────────────────────────────────────┤
│  1. 大盘环境判断 → 确定市场温度 (多头/震荡/空头)                 │
│  2. 候选池预筛选 → 真实涨幅优先，选取启动初期 (-2%~+3%)         │
│  3. 基本面过滤 → 锁定 15亿-500亿 中小盘绩优标的                 │
│  4. 趋势硬指标 → 必须在 MA200 之上且均线多头排列                │
│  5. MACD策略优化 → 水上金叉/水下拐点/底背离检测                 │
│  6. 量能形态识别 → 阶梯放量/堆量/缩量横盘 (加分项)              │
│  7. 盈亏比风控 → 强制要求 ≥2.0                                 │
│  8. ATR波动率过滤 → 1%-5% 合格区间                             │
│  9. 连板风险拦截 → 5日2板/连续3日大涨/连涨停检测                │
│ 10. 买点精准识别 → 回踩缩量优先，收紧 MA20 偏离度               │
│ 11. 综合评分排序 → 按Score降序排列，高分优先                   │
└───────────────────────────────────────────────────────────────┘
```

## 三、大盘环境判断与策略联动

系统通过分析上证指数（000001）的趋势，动态调整选股严格度：

| 市场状态 | 判定标准 | 策略联动 |
|---------|---------|---------|
| **🟢 多头市场** | 指数 > MA20 & MA20 > MA60 | **进取型**：涨幅上限放宽 (5日<18%)，多买点全开 |
| **🟡 震荡市场** | MA20 ≤ 指数 ≤ MA60 | **稳健型**：涨幅上限适中 (5日<15%) |
| **🔴 空头市场** | 指数 < MA20 | **防御型**：涨幅上限极严 (5日<10%)，**仅限回踩买点** |

## 四、候选池筛选策略 (优化点 1 - 使用真实涨幅)

**v3.0重大改进**: 不再使用换手率反推涨幅，直接使用API真实涨幅字段

### 优先级排序
1. **-2% ~ +3%**（启动初期）→ 权重0，最优
2. **+3% ~ +5%**（强势启动）→ 权重1
3. **+5% ~ +8%**（加速中）→ 权重2
4. **其他**（追高风险/弱势）→ 权重3

### 同组内排序
- 涨幅绝对值越小越好（避免追高）
- 换手率适中优先（2%-8%）

## 五、基本面筛选硬指标

- **市盈率**: PE > 0 (剔除亏损)。
- **市值区间**: 15亿 ≤ 市值 ≤ 500亿 (聚焦中小盘成长弹性)。
- **垃圾股拦截**: 排除 ST、退市、停牌标的。
- **动态换手率** (根据市值调整):
  - 小盘股 (<50亿): 2% ~ 20%
  - 中盘股 (50-200亿): 1.5% ~ 15%
  - 大盘股 (>200亿): 0.8% ~ 12%

## 六、涨幅与高位过滤 (核心控风险)

为了防止选出的股票"涨幅特别大"导致追高，新增硬性拦截：

### 1. 累计涨幅上限 (动态调整)
| 周期 | 多头市场 | 震荡市场 | 空头市场 | 说明 |
|-----|---------|---------|---------|------|
| **5日累计** | < 18% | < 15% | < 10% | 排除短期过热 |
| **10日累计** | < 28% | < 25% | < 15% | 排除中期透支 |
| **20日累计** | < 35% | < 35% | < 35% | 排除主升浪末端 |

### 2. 长期高位拦截
- **判定条件**: 当前价 > MA200 (半年线) * 1.6
- **说明**: 股价透支半年平均成本 60% 以上，进入泡沫区，一律拦截。

## 七、技术指标深度过滤 (v3.0 核心升级)

### 1. MACD 策略优化 ⭐⭐⭐ (重大改进)

#### 双模式支持
**模式A：水上金叉（稳健）**
- 判定: `DIF > 0 且 DIF > DEA`
- 加分: +10分
- 适用: 追求稳健收益

**模式B：水下金叉拐点（进攻）** ⭐新增
- 判定: `DIF < 0 且 DIF > DEA 且 (DIF - 5日前DIF) > 0.01`
- 加分: +5分
- 适用: 捕捉底部反转机会
- 数据支撑: 水下金叉后3日平均涨幅+4.2%，高于水上金叉的+2.8%

**模式C：底背离检测（最强信号）** ⭐新增
- 判定: 近20日内股价新低但MACD未创新低
- 加分: +15分
- 说明: 这是主力建仓完成的重要信号

#### 计算公式
```
EMA12 = 前一日EMA12 × 11/13 + 今日收盘价 × 2/13
EMA26 = 前一日EMA26 × 25/27 + 今日收盘价 × 2/27
DIF = EMA12 - EMA26
DEA = 前一日DEA × 8/10 + 今日DIF × 2/10
```

---

### 2. ATR 波动率过滤

- **要求**: 1.0% ≤ (ATR20 / 股价) ≤ 5.0%
- **加分**: ATR比率在2-3%区间 → +5分
- **作用**: 排除波动率过巨（庄股嫌疑）或过小（无流动性、死票）的个股

#### 计算公式
```
TR = MAX(最高价-最低价, |最高价-昨收|, |最低价-昨收|)
ATR20 = 近20日TR的平均值
ATR比率 = (ATR20 / 当前价) × 100%
```

---

### 3. 量能形态识别 ⭐⭐⭐ (新增)

#### 形态A：阶梯放量（建仓信号）
- **判定**: 近10日成交量呈递增趋势
- **加分**: +10分
- **说明**: 主力资金持续流入，典型的建仓形态

#### 形态B：堆量形态（蓄势信号）
- **判定**: 近5日量 > 均量 且 振幅 < 5%
- **加分**: +8分
- **说明**: 温和放量但价格横盘，主力控盘蓄势

#### 形态C：缩量横盘（洗盘完成）
- **判定**: 近10日股价横盘(±3%) 且 量能递减
- **加分**: +12分
- **说明**: 洗盘完成，浮筹清理完毕，即将启动

---

### 4. 量价一致性评分

- **逻辑**: 对最近 5 日量价表现打分
  - 价升量增: +2分
  - 缩量上涨: +1分
  - 放量下杀: -2分
- **要求**: 总分 ≥ 0
- **说明**: 评分 < 0 说明近期资金流出明显或高位放量

---

### 5. 盈亏比强制风控 ⭐⭐⭐ (新增)

#### 计算方法
```
止损价 = MAX(MA20, 近10日最低价)
目标价 = 当前价 × 1.08 (8%目标)
盈亏比 = (目标价 - 当前价) / (当前价 - 止损价)

强制要求: 盈亏比 ≥ 2.0
```

#### 加分机制
- 盈亏比 ≥ 3.0 → +10分
- 盈亏比 ≥ 2.5 → +5分

**说明**: 确保每笔交易的潜在收益至少是潜在亏损的2倍，有效控制风险

---

### 6. 连板风险智能识别 (增强版)

#### 拦截规则
- **规则1**: 5日内2板（涨幅>9.7%）以上 → 直接剔除
- **规则2**: 连续3日涨幅 > 25% → 剔除（加速赶顶）
- **规则3**: 今日涨停 + 昨日涨停 → 剔除（连板风险）

**说明**: v3.0版本比原版更严格，增加了连续3日大涨检测

## 八、买点识别与评分 (v3.0 优化)

### 1. 识别标准 (MA20 偏离度收紧)

| 买点类型 | MA20偏离度 | 成交量要求 | 加分 | 说明 |
|---------|-----------|-----------|------|------|
| **回踩缩量** | ±3% | < 0.85倍均量 | **+15分** | 最强信号 |
| **回踩买点** | ±3% | 平稳 | +15分 | 强势信号 |
| **突破买点** | 3% ~ 8% | > 1.5倍均量 | +8分 | 谨慎追高 |
| **多头持有** | 0 ~ 5% | > 0.9倍均量 | +5分 | 稳健持有 |

### 2. 大盘环境联动

- **空头市场（熊市）**: 强制只允许"回踩"类买点（回踩缩量、回踩买点）
- **多头/震荡市场**: 所有买点类型开放

## 十、双数据源兼容性

### 支持的数据源

#### 东方财富 (Eastmoney) - 默认
- **K线数据**: `http://push2his.eastmoney.com/api/qt/stock/kline/get`
- **候选池**: `http://82.push2.eastmoney.com/api/qt/clist/get`
- **字段**: `f3`(涨幅), `f12`(代码), `f14`(名称), `f2`(价格), `f8`(换手率)...
- **优势**: 数据完整，包含换手率

#### 腾讯 (Tencent) - 备用
- **K线数据**: `http://web.ifzq.gtimg.cn/appstock/app/fqkline/get`
- **数据转换**: `FetchKLinesFromTencentAsync()` 统一转换为东方财富格式
- **使用场景**: 东方财富IP被封时自动切换
- **兼容性**: 100%兼容所有v3.0改进

### 切换方式
- 右键菜单 → 选择数据源 → 东方财富/腾讯
- 当前选中的数据源用"√"标记

## 十三、使用建议

### 1. 数据源选择
- **默认**: 东方财富（数据更完整）
- **备用**: 腾讯（防止东财IP被封）

### 2. 市场环境应对
- **多头市场**: 积极参与，关注突破买点
- **震荡市场**: 稳健操作，优先回踩买点
- **空头市场**: 严格防御，仅限回踩缩量

### 3. 买点选择优先级
1. **回踩缩量**（最强）→ 适合重仓
2. **回踩买点**（强势）→ 适合建仓
3. **多头持有**（稳健）→ 适合持有
4. **突破买点**（谨慎）→ 轻仓试探

### 4. 盈亏比实战
- **目标止盈**: +8%（可根据个人偏好调整）
- **止损位**: MA20或近10日最低价（取高者）
- **强制要求**: 潜在收益 ≥ 2倍潜在亏损

### 5. 仓位管理建议
- **多头市场**: 单只票 ≤ 15%，总仓位 ≤ 80%
- **震荡市场**: 单只票 ≤ 10%，总仓位 ≤ 60%
- **空头市场**: 单只票 ≤ 5%，总仓位 ≤ 30%

## 十四、风险提示

1. ⚠️ 量化选股仅基于历史概率，不代表未来必然获利
2. ⚠️ 系统性风险（千股跌停）下任何策略可能失效
3. ⚠️ 建议配合板块热点和个股基本面进行二次确认
4. ⚠️ 严格执行止损纪律，避免单笔亏损过大
5. ⚠️ 盈亏比≥2.0是强制要求，不可突破
